** How to interact with the Calendar **

When the user asks for information about their calendar (make an appointment, what their day is like, etc) you can use AppleScript to talk to Mac OS Calendar app.  There is a tool call "run_applescript" that you can use to invoke the script that you wrote.

When you create calendar events, you have to tell Calendar the exact start and times.  Here's an example script you can base your work on:

```applescript
set theStartDate to (current date) + (1 * days)
set hours of theStartDate to 15
set minutes of theStartDate to 0
set seconds of theStartDate to 0
set theEndDate to theStartDate + (1 * hours)
 
tell application "Calendar"
    tell calendar "Project Calendar"
        make new event with properties {summary:"Important Meeting!", start date:theStartDate, end date:theEndDate}
    end tell
end tell
```

** How to interact with Email **

When the user asks you to interact with Email know that they use Mac OS X's default Mail app.  You can talk to it with AppleScript using the run_applescript tool.  Here is an example script to get the subject line of the mail at the top of their inbox:

```applescript
tell application "Mail"
    set firstMessage to first message of inbox
    set subjectLine to subject of firstMessage
    return subjectLine
end tell
```

Here's a more indepth example for interacting with Mail.  This example retrieves the most recent 5 messages from an account.

```applescript
tell application "Mail"                                                                                 
    set msgList to {}
    set msgCount to 0
    try
        tell mailbox "INBOX" of account "myname@account.net"
            set msgCount to count of messages
            repeat with i from 1 to 5
                if i <= msgCount then
                    set m to item i of messages
                    set s to subject of m
                    set d to date received of m
                    set end of msgList to "Subject: " & s & ", Date: " & d
                end if
            end repeat
        end tell
        end try
    return "Count: " & msgCount & " Messages: " & msgList
end tell                                                                                                
```

This AppleScript can retrieve the latest email from your oskin@cs.washington.edu inbox:

```applescript
tell application "Mail"
    -- Check if Mail is actually ready
    if not (exists account "oskin@cs.washington.edu") then
        return "Account not found."
    end if
    
    set theAccount to account "email@domain.net"
    set theMailbox to mailbox "INBOX" of theAccount
    
    -- Try to get a simple property first to verify connection
    try
        set msgCount to count of messages of theMailbox
        if msgCount is 0 then
            return "Inbox is empty."
        end if
        
        -- Get the first message
        set latestMessage to first message of theMailbox
        
        -- Get basic properties which usually don't trigger -1741
        set theSubject to subject of latestMessage
        set theSender to sender of latestMessage
        set theDate to date received of latestMessage
        
        -- Try to get the text version of the body
        set theBody to ""
        try
            -- Some versions of Mail support 'text' property directly on the message
            set theBody to text of latestMessage
        on error
            try
                -- Fallback to content
                set theBody to content of latestMessage
            on error
                set theBody to "Body content could not be retrieved due to permission/access restrictions."
            end try
        end try
        
        return "Subject: " & theSubject & "\nFrom: " & theSender & "\nDate: " & theDate & "\n\n" & theBody
        
    on error e
        return "Error: " & e
    end try
end tell
```

** How to internact with Notes **

Here is the AppleScript I used to create the TODO list note with proper bullet points:

To interact with notes you send properly formatted HTML.  Here's an example that creates a TODO list.

```applescript
tell application "Notes"
     -- Find the first note named "TODO List" and delete it
    set theAccount to account 1
    set allNotes to every note of theAccount
    repeat with i from (count of allNotes) to 1 by -1
        set thisNote to item i of allNotes
        if name of thisNote is "TODO List" then
            delete thisNote
        end if
    end repeat
    
     -- Create the new note with proper HTML formatting for bullet points
    set htmlBody to "<ul>\n<li>Breakfast with Wayne</li>\n<li>reschedule Marcie</li>\n<li>Finish agent</li>\n</ul>"
    set newNote to make new note with properties {name:"TODO List", body:htmlBody}
end tell
```

And here is a script that shows how one adds an entry to an existing note:

Here is the AppleScript I used to add "Make dinner" to the "TODO List" note:

```applescript
tell application "Notes"
    set theAccount to account "iCloud"
    set allNotes to every note of theAccount
    set targetNote to missing value
    repeat with i from 1 to count of allNotes
        set thisNote to item i of allNotes
        if name of thisNote is "TODO List" then
            set targetNote to thisNote
            exit repeat
        end if
    end repeat
    
    if targetNote is missing value then
        error "TODO List note not found"
    end if
    
    set currentBody to body of targetNote
    
    set AppleScript's text item delimiters to "</ul>"
    set bodyParts to text items of currentBody
    set AppleScript's text item delimiters to ""
    set newBody to (item 1 of bodyParts) & "<li>Make dinner</li>\n</ul>"
    
    set body of targetNote to newBody
end tell
```

** How to interact with Reminders **

To show what Reminder lists are available:

```applescript
tell application "Reminders"
    set listNames to {}
    repeat with theList in lists
        set end of listNames to name of theList
    end repeat
    return listNames
end tell
```

To add an entry to a reminder list:

```applescript
tell application "Reminders"
    set theList to list "Work Items"
    make new reminder in theList with properties {name:"write lecture for monday"}
end tell
```

To list items in a Reminder list:

```applescript
tell application "Reminders"
    set theList to list "Work Items"
    set itemNames to {}
    repeat with theReminder in theList
        set end of itemNames to name of theReminder
    end repeat
    return itemNames
end tell
```

To check off a reminder:

```applescript
tell application "Reminders"
    set theList to list "Work Items"
    repeat with theReminder in theList
        if name of theReminder is "attend talk" then
            set completed of theReminder to true
            exit repeat
        end if
    end repeat
end tell
```

** To interact with iTunes **

Here is the AppleScript I used to list the playlists in the Music app:

```applescript
tell application "Music"
    set playlistNames to name of every playlist
    return playlistNames
end tell
```

To play a playlist

```applescript
tell application "Music"
    set myPlaylist to playlist "Mixtape"
    play myPlaylist
end tell
```
